가톨릭 성가책 관리 계획 보고

(엠마오 사목회의, `24. 4. 27)

■ 관리주관 부서 : 전례분과
■ 현황 파악 : `24. 4. 20(토) 특전 미사후
■ 확인 결과
● 보유 수량
① 현재 사용하는 성가책 : 125권(가톨릭 성가 수정판)
※ 유아실 보관 성가 : 가톨릭 성가책(`22년도 수정판 출간으로 현재 미사용)
② 1개 탁자당 4권씩 비치시 18권 부족
● 상 태
① 성가책 1차 스카치테이프로 보수된 상태
② 내부 훼손된 성가책은 없음
③ 대부분 겉장 접히는 부분이 훼손
④ 재보수시 더 지저분한 상태로 사용될 우려 있음
● 성가책 소요/예산 판단 : 1개 탁자당 4권 비치시 150권 소요(평시)
※ 예산 : @15,000, 225만원
■ 판 단
● 방안 1) : 보수하여 사용
※ 예산 절감 효과 있으나 사용 가능 기간 단축으로 6개월 이내 구입 시기 도래 예상
● 방안 2) : 성당 예산 편성하여 구입
※ 년간사업계획에 미포함된 사업 추진
● 방안 3) : 교우 대상으로 무기명 찬조로 구입
※ 전 교우 참여를 통한 동참 의식 부여
■ 건 의
● 방안 3) : 교우 대상으로 무기명 찬조로 구입
① 교우 부담을 최소화하기 위해 주보에 2회정도 게재후 모금된 금액만큼 구입하고 부족한 수량은 현재 보유한 성가책 보수하여 사용
② 대림 및 사순시기 미사참여가 많을 때를 대비하여 보유 성가책 모두 보수후 보관
● 성가책 종류 :
① 규격/단가 : 148*220 @18,000원(헌금, 성체 분배시 걸림 현상 예상)
② 규격/단가 : 128*189 @15,000원
